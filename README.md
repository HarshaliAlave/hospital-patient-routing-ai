# 🏥 Hospital Patient Routing System — Agentic AI

An AI-powered patient routing system built with LangGraph + Groq LLM.

## 🔧 Tech Stack
- LangGraph (StateGraph)
- LangChain-Groq (LLM)
- Python + Pandas
- Google Colab

## 🚀 How it works
1. Patient enters name, age & symptoms
2. Groq LLM classifies symptoms
3. Patient routed to correct ward (Emergency / General / Mental Health)
4. System checks doctor availability in real-time
5. Doctor assigned or patient queued

## 📊 Wards
- 🚨 Emergency Ward
- 🏥 General Ward  
- 🧠 Mental Health Ward

## 🏃 How to run
1. Open in Google Colab
2. Install dependencies: pip install langgraph langchain-groq pandas
3. Add your Groq API key
4. Upload doctors.csv
5. Run all cells!
